# Sitio Web - Óptica SOONAHERBAL
**Sitio web profesional de presentación para una óptica. Página informativa moderna y responsive donde los clientes pueden conocer servicios, ver productos y contactar el negocio.**
# Características Principales
- Menú de navegación fijo con 5 secciones (Inicio, Acerca, Servicios, Productos, Contacto)
- Portada visual impactante con imagen de fondo y texto superpuesto
- Sección de contenido principal con presentación detallada
- Barra lateral informativa con artículos destacados
- Pie de página profesional con logo y redes sociales
# Diseño Responsivo Avanzado
- Menú adaptativo que se transforma en "hamburguesa" en móviles
- Layout flexible que se ajusta automáticamente a pantallas
- Transiciones suaves para una experiencia de usuario fluida
- Imágenes optimizadas para carga rápida
# Tecnologías Utilizadas
- HTML5
- CSS3
- JavaScript
- Font Awesome
# Diseño Visual Profesional
- Paleta de colores: Verde natural (#012201) con fondos claros
- Tipografía elegante: 'Playfair Display' para títulos
- Efectos visuales: Sombras, transiciones y hover states
- Espaciado consistente y jerarquía visual clara
